Dirichlet Convolution

If f,g:N→C are two arithmetic functions from the positive integers to the complex numbers, the Dirichlet convolution f ∗ g is a new arithmetic function defined by:

(f∗g)(n) = d∣n∑​f(d)g(dn​) = ab=n∑​f(a)g(b)
  • Commutative: f∗g=g∗f
  • Associative: (f∗g)∗h=f∗(g∗h)
  • Distributive over pointwise addition: f∗(g+h)=f∗g+f∗h
    • Pointwise addition: f+g is defined by (f+g)(n)=f(n)+g(n)
  • Multiplicative Identity: f∗ε=ε∗f=f
    • ε is the unit function defined as
      ε(n)={1,0,​if n=1if n=1​
  • Multiplicative Inverse: for each f having f(1)=0, there exists an arithmetic function f−1 with f∗f−1=ε, called the Dirichlet inverse of f

Important functions​

  • ε - Unit function

  • μ - Mobius function

  • 1∗μ=ε
