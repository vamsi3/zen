문법​

~께서 - 명사​

간략한 설명

~께서 - this is in place of 이/가. Formal way to address senior people, including parents or old people.

  • 할아버지께서 신문을 읽으십니다.

~에게/께 - 명사​

간략한 설명

This means 'To someone'. ~께 is for older people, including parents.

  • 선물을 부모님께 드렸어요.

~(으)ㄹ까요? - 동사​

간략한 설명

This is a polite way of asking to do something together. Similar to, 'Shall we...?'.

~(으)ㅂ시다 - 동사​

간략한 설명

This is a polite way to agree with a suggestion.

    • 영호: 우리 영화를 볼까요?
    • 미화: 좋아요. 영화를 봅시다.

~(이)지요? - 명사​

간략한 설명

This is used to suggest an answer, like 'Right?'. Sometimes shortened to '죠'.

  • 여보세요? 거기 마리아 씨 집이지요?

~(으)러 가다/오다/다니다 - 동사​

간략한 설명

This means, 'To go/to come/to regularly go to...' in that order. It is used for present or future tense.

  • 약국에 약을 사러 가요.

~지 말다 - 동사​

~(으)ㄴ + 명사 - 동사 / ~는 + 명사 - 동사 / ~(으)ㄹ + 명사 - 동사​

~(으)ㄴ + 명사 - 형용사​

문법 (AK)​

[동사] ~게 되다​

note

This is used to describe how something or someone comes to be in or becomes a certain state or how something happens, often by chance or fate.

  • https://www.koreanwikiproject.com/wiki/게_되다

[형용사] ~아/어 보이다​

note

Used to guess or express the perception of a person or thing at a glance.

예: 넌 기분이 안 좋아 보이네.

[동사] ~지 그래요?​

note

To ask or suggest someone politely to do something.

[동사] ~지 그래요? feels stronger than [동사] ~는 게 어때요?

[동사] ~자고 하다​

note

Indirect speech of a propositive sentence. (Recall that in 반말, propositive sentences end with ~자)

Negative version: [동사] ~지 말자고 하다

Usually applicative to,

  • ~자.
  • ~(으)ㅂ시다.
  • ~(으)ㄹ까요?
  • ~(으)ㄹ래요?
  • ~아/어요.

[동사] ~(으)라고 하다​

note

Indirect speech of a imperative sentence. (meaning, order or command)

Negative version: [동사] ~지 말라고 하다

But for changing N을/를 주세요 to indirect speech, N을/를 주라고 하다 cannot be used when receiver is the speaker himself. In that case use N을/를 달라고 하다.

[동사] ~느냐고 하다 / [형용사] ~(으)냐고 하다 / [명사] ~(이)냐고 하다​

note

Indirect speech of a interrogative sentence.

Colloquial -> [동사/형용사] ~냐고 하다 (i.e. 느 or 으 is dropped often)

  1. Present tense: [동사] ~느냐고 하다 / [형용사] ~(으)냐고 하다 / [명사] ~(이)냐고 하다
  2. Past tense: [동사/형용사] ~았/었느냐고 하다 / [명사] ~이었/였냐고 하다 --- (명사 part not sure)
  3. Future tense: [동사/형용사] ~(으)ㄹ 거냐고 하다
  • Instead of 하다, 묻다 or 물어보다 is also common.
  • 있으세요? -> 있으시냐고 하다

불규칙​

'ㅎ' 불규칙​

  1. ~아/어 -> ~애 / ~야 -> ~얘 (+ ㅎ removed)
    • 예: 노랗다 + 아요 = 노래요
  2. ~(으)~X -> X (+ ㅎ removed)
    • 예: 노랗다 + (으)ㄴ = 노란

자잘한 구절​

Clothes​

  • 입다 -> 치마, 비지, 셔츠
  • 신다 -> 구두, 운동화, 양말
  • 쓰다 -> 모자, 안경, 우산
  • 매다 -> 스카프, 넥타이
  • 끼다 -> 인경, 빈지
  • 치다 -> 시계
  • 하다 -> 귀걸이, 목걸이, 머리띠
  • 들다 -> (holding in hand, example 가방)
  • 메다 -> (carrying over shoulder), example 가방)

Other​

  • 평소처럼 해요 -> Do as usual

    • 평소 -> usual [ordinary, average, regular] day
    • 평소처럼 -> as usual

  • 편지를 부치다 -> send a letter.

  • 비자를 연장하다 -> Prolong/Extend a visa.

  • 우산을 가지고 가다 -> Carry/Take an umbrella. (with you)

  • 전화를 걸다 -> to make a (phone) call

  • 너래를 부르다 -> to sing a song

  • ABC~하고 닮았어요 -> to resemble ABC

질문​

  • 작다 vs 적다
