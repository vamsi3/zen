Number Of Divisors

Question 1#

Question

Given n≤1018, find number of divisors of n.

Resources
  • Codeforces Tutorial: https://codeforces.com/blog/entry/22317
  • Codeforces Gym: https://codeforces.com/gym/100753 ➝ Problem K
  • SPOJ: https://www.spoj.com/problems/NUMDIV/

Naive Solution#

The complexity of the following is O(n​).

Naive Solution
void solve() {
int n; cin >> n;
int an = 0;
for (int i = 1; i * i <= n; ++i) {
if (n % i == 0) {
an += 2;
if (i * i == n) --an;
}
}
cout << an << '\n';
}

O(n31​) solution#

  • Let's split n=x×y where x contains only prime factors in range [2,n31​] and rest of primes in y.
  • Let f(n) denote our answer. Now, since x and y are coprime, f(n)=f(x)×f(y).
  • f(x) can be calculated directly similar to naive solution in O(n31​).
  • Observe that y falls into one of these cases:
    • y is prime ⟹ f(y)=2
    • y is square of prime ⟹ f(y)=3
    • y is product of two distinct primes ⟹ f(y)=4
  • This is because there can be at max only two divisors of y, otherwise one of them surely must be ≤n31​.
//pseudocode -- will update later
void solve() {
int n; cin >> n;
vector<int> pr; // all primes till 10^6
int an = 1;
for (auto &p : pr) {
if (p * p * p > n) break;
int cn = 1;
while (n % p == 0) { n /= p; ++cn; }
an *= cn;
}
// use miller-rabin
if (n is prime) { an *= 2; }
else if (n is square of prime) { an *= 3; }
else an *= 4;
cout << an << '\n';
}
